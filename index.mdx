Scada-LTS is an open-source Supervisory Control and Data Acquisition (SCADA) system built for industrial environments. It connects to industrial protocols like Modbus, BACnet, and SNMP, lets you build real-time graphical views of your processes, and provides alarm management with email and SMS notifications — all through a web browser.

## Get started in three steps

<Steps>
  <Step title="Deploy with Docker">
    Pull the official image and start Scada-LTS with a single Docker Compose command. MySQL is included in the compose file.

    ```bash
    docker compose up -d
    ```
  </Step>
  <Step title="Log in and add a data source">
    Open your browser at `http://localhost:8080/Scada-LTS` and log in with the default credentials. Navigate to **Data Sources** and add your first Modbus, BACnet, or virtual data source.
  </Step>
  <Step title="Build a graphical view">
    Go to **Graphical Views**, create a new view, and drag data point components onto your process background image to build a live SCADA display.
  </Step>
</Steps>
